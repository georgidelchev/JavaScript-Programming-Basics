## 📚 What This Course Covers

The JavaScript Basics course introduces programming fundamentals through one of the world's most widely used languages:

- **Console I/O** — reading input and writing output in a JavaScript runtime environment
- **Data types & variables** — numbers, strings, booleans, and dynamic typing in JS
- **Operators & expressions** — arithmetic, comparison, and logical operations
- **Conditional statements** — `if/else`, nested conditions, and `switch`
- **First steps in problem solving** — translating real-world problems into working code

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/georgidelchev/JavaScript-Programming-Basics.git
   cd JavaScript-Programming-Basics
   ```

2. **Open any `.js` file** in VS Code or your preferred editor.

3. **Run a solution** with Node.js:
   ```bash
   node solution.js
   ```
